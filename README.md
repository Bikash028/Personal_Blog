# Personal Blog Website

A full-stack blogging platform built using Flask, SQLite, Bootstrap, and Flask-Login.

## Features

- User Registration
- User Login & Logout
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- View All Blogs
- View Individual Blog
- Responsive UI
- SQLite Database

## Technologies Used

### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

### Backend
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-WTF

### Database
- SQLite

## Installation

### Clone Repository

git clone https://github.com/yourusername/cognevance_personalBlog.git

cd cognevance_personalBlog

### Create Virtual Environment

python -m venv venv

### Activate Environment

Windows:

venv\Scripts\activate

Linux/Mac:

source venv/bin/activate

### Install Dependencies

pip install -r requirements.txt

### Run Project

python app.py

Open:

http://127.0.0.1:5000
